# Code of Conduct

## What this project is about

AetherSuite is a technical project built for everyone. People of all backgrounds, skill levels, and ages use and contribute to it. This code of conduct exists to make sure the experience stays positive for all of them.

## What is expected

Be respectful. When you disagree with someone, address the technical point without attacking the person.

Be patient. Not everyone has the same experience level. Someone asking a basic question deserves a clear answer, not frustration.

Be constructive. If you criticise something, explain what would be better and why.

Stay on topic. Keep discussions focused on the project.

## What is not accepted

Personal attacks, insults, and harassment have no place here. This applies to issues, pull requests, comments, and any other space connected to this project.

Discrimination based on age, gender, nationality, religion, skill level, or any other personal characteristic is not tolerated.

Spam and off-topic self-promotion will be removed.

## What happens when something goes wrong

If you see behaviour that breaks these rules, report it by opening a private message to the maintainers. We will look at it and act accordingly, up to and including removing the person from the project.

If you are unsure whether something crosses a line, ask yourself : would I say this to someone's face in a professional setting ? If not, do not write it here.

---

This code of conduct applies to everyone who participates in this project, regardless of their role.
