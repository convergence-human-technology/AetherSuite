# Changelog

All notable changes to AetherSuite are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026

### Added

- AetherSMS : SMS and MMS messaging with smart photo compression, PDF attachments, voice-to-text in French and English, clipboard button, automatic spelling correction, conversation search, and automatic draft saving.
- AetherContacts : full contact management with favourites, search, and direct integration with AetherSMS and AetherPhone.
- AetherPhone : phone dialer with call log and one-tap contact saving to AetherContacts.
- AetherNotes : encrypted notes using AES-256-GCM, colour labels, and grid view.
- AetherFiles : file manager with navigation, sorting, and one-tap note creation from any file.
- AetherGallery : photo and video gallery with full-HD sharing to AetherSMS.
- AetherMusic : local music player supporting MP3 and FLAC, with background playback.
- AetherCalendar : local encrypted calendar with AlarmManager reminders and SMS event sharing.
- aether-core : shared design system (AetherTheme, AetherIntents), AES-256 encryption layer, and biometric support.
- Full inter-app communication across all 9 applications.
- Light and dark themes only, no other visual modes.
- Zero ads, zero telemetry, zero network calls for messaging.
